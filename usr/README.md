# Professional Investment Tracker

A modern, responsive Investment Tracker built with Flutter. It allows individuals to monitor and grow their personal investments. The app features a clean, minimalist layout with a professional color palette (navy, green, black, and white) and adapts seamlessly to mobile, tablet, desktop, and web environments.

## Features

*   **Portfolio Dashboard:** View overall portfolio performance, total invested, current value, total profit/loss, and total return.
*   **Investment Details:** Track individual investments including:
    *   Asset Name & Ticker
    *   Investment Type (Stocks, ETFs, Crypto, etc.)
    *   Platform/Broker
    *   Shares/Units & Purchase Price
    *   Current Market Price
    *   Total Amount Invested & Current Value
    *   Profit/Loss ($ and %)
    *   Dividends/Interest
    *   Total Return & Risk Level
    *   Portfolio Allocation
*   **Responsive Design:** Optimized layout for both narrow mobile screens (list views, stacked cards) and wide desktop/web screens (data tables, side-by-side panels).

## Tech Stack

*   Flutter (Dart)
*   Material Design 3

## Setup & Run

To run the application locally:

1.  Ensure you have Flutter installed.
2.  Clone this repository.
3.  Run `flutter pub get` to install dependencies.
4.  Run `flutter run` to launch the app on your preferred device or emulator.
